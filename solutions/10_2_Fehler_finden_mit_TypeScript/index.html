10.2. Fehler finden mit TypeScript

Fehlerhafter Code:

        const add = (x,y) => x+y;
        const equals = (x,y) => x===y;

        console.log( add(1,2) );
        console.log( add(true, true ) );
        console.log( add(true, false ) );
        var x = y = 1;
        console.log( add(equals(x,y), equals(y,x)) );

Compiler-Fehler:

  • Parameter 'x' implicitly has an 'any' type.(7006)
  • Parameter 'y' implicitly has an 'any' type.(7006)
  • Cannot find name 'y'.(2304)

Korrektur:

        const add = (x:number,y:number) => x+y;
        const equals = (x:any,y:any) => x===y;

        console.log( add(1,2) );
        console.log( add(true, true ) );
        console.log( add(true, false ) );
        var y:number;
        var x:number = y = 1;
        console.log( add(equals(x,y), equals(y,x)) );

Compiler-Fehler:

  • Argument of type 'boolean' is not assignable to parameter of type 'number'.(2345)
